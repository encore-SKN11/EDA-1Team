## 🧹 전처리 (Preprocessing)

### ✅ 기본 전처리

- 결측치 확인 및 처리
- `골든글러브` → float(0.0 또는 1.0) 로 변환 (회귀용)

### ✅ 리그 환경 반영

- `타율_차이 = 타율 - 리그_평균_타율`
- `OPS_차이 = OPS - 리그_평균_OPS`
- `리그_환경` → One-Hot Encoding (예: 타고투저, 투고타저, 중립)

### ✅ 범주형 처리

- 팀 → One-Hot Encoding
- 연도는 수치 그대로 두거나 One-Hot Encoding (선택)

### ✅ 정규화

- `StandardScaler` 또는 `MinMaxScaler`로 수치형 피처 스케일링 (선형 회귀 성능 향상)
